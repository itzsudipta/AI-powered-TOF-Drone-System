## 📌 Project Overview
The **AI-Powered TOF Drone System** is a cutting-edge research and development project focused on enhancing drone perception and analytics. Leveraging advanced computer vision models like YOLOv8 combined with Ensemble Learning, this system pushes the boundaries of object detection and spatial awareness. 

> **⚠️ Note:** This project is currently under active development and is directly related to an upcoming/ongoing **patent application**.

---

## 🚀 Current Status & Features
The project has reached a stable testing phase with the following milestones achieved:

- 🧠 **High-Accuracy AI:** The model currently utilizes **Ensemble Learning** and is performing exceptionally well with an accuracy score of **0.84 (84%)**.
- 🖥️ **Full-Stack Integration:** A robust **Flask backend** has been successfully integrated with our custom HTML/CSS/JS frontend.
- 🌐 **Live Deployment:** Temporary live server deployment has been established using **ngrok** for testing and demonstration purposes.
- 📚 **Extensive Documentation:** Comprehensive project documentation has been added to guide further development and usage.

## 🛠️ How It Works (Architecture)
1. **Model Training:** YOLOv8 models trained and optimized via Google Colab.
2. **Ensemble Learning:** Multiple model predictions are aggregated to achieve the 0.84 accuracy threshold.
3. **Backend Processing:** Flask handles routing, API requests, and serves the machine learning predictions.
4. **User Interface:** A clean, responsive Frontend built with HTML, CSS, and JS displays the drone data to the user.
5. **Tunneling:** Ngrok exposes the local Flask server to the internet for remote access.
